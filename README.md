# Pacman
University project, features:

- Maze generator
- Path finding mode
- Auto mode (almost AI, but it is not. Tree-search basically)
- Average-quality code :)
- Nice animations
- Minecraft styled map and ghosts
- Different color schemes
- Different pathfinding algorithms
